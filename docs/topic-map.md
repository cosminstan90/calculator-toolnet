## Regula pentru tot topic map-ul

Fiecare cluster trebuie sa aiba:

1. un hub
2. 3-5 calculatoare `tier-1`
3. 2-4 articole `tier-1`
4. legaturi clare intre ele
5. un traseu bun pentru:
   - utilizator
   - Google
   - AI overviews / LLM citations
